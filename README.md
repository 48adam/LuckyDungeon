# LuckyDungeon

**LuckyDungeon** to hackathonowy projekt gry 3D stworzony w **Unity**, którego główną mechaniką jest **zarządzanie czasem**.  
Gracz może poświęcać czas pozostały do ukończenia poziomu, aby ułatwić sobie przejście danego etapu, co dodaje rozgrywce element strategii i ryzyka.

## O grze

Projekt powstał podczas konkursu programistycznego typu **hackathon**.  
Założeniem było stworzenie gry komputerowej w krótkim czasie, z wykorzystaniem darmowych narzędzi i publicznie dostępnych zasobów, takich jak modele 3D.

LuckyDungeon to prototyp gry 3D, który skupia się na ciekawej mechanice podejmowania decyzji - gracz musi wybrać, czy warto „zapłacić czasem”, by zwiększyć swoje szanse na ukończenie poziomu.

## Najważniejsze cechy

- gra 3D stworzona w Unity
- oryginalna mechanika handlu czasem
- projekt wykonany zespołowo podczas hackathonu
- gotowa wersja do uruchomienia na Windows
- wykorzystanie darmowych assetów i narzędzi

## Jak uruchomić

1. Pobierz repozytorium
2. Zachowaj wszystkie pliki w jednym folderze
3. Uruchom plik `Lucky Dungeon.exe`

## Wymagania

- Windows
- komplet plików builda Unity:
  - `Lucky Dungeon.exe`
  - `UnityPlayer.dll`
  - `Lucky Dungeon_Data/`
  - `MonoBleedingEdge/`

## Technologie

- **Unity**
- **C#**
- assety 3D z darmowych źródeł

## Struktura projektu

- `Lucky Dungeon.exe` - plik uruchomieniowy gry
- `Lucky Dungeon_Data/` - zasoby i dane gry
- `MonoBleedingEdge/` - środowisko uruchomieniowe
- `UnityPlayer.dll` - biblioteka Unity
- `UnityCrashHandler64.exe` - obsługa błędów
- `D3D12/` - komponenty związane z renderowaniem

## Status projektu

Projekt stanowi działający prototyp przygotowany na potrzeby hackathonu.  
Repozytorium zawiera obecnie build gry, bez pełnych plików źródłowych Unity.
